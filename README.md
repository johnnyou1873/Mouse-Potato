# Mouse-Potato
Micromouse design for AAMC 2024

## Components

### Microcontroller
STM32F411RETx

### Motors
CHF-GW12T-N20VA ABHL DC Worm Gear Motor
- Ratio: 42-A
- Voltage: 6V B45
- https://www.aliexpress.us/item/3256801422029348.html
- Requires shortening of the encoder board and resoldering of the cables

### IMU
ICM-42688-P
